Installing Video Add-On

First enable Kodi to download 3rd Party Add-Ons

  1. In the Kodi App click on the gear icon(top left) to enter the Systems Menu
  2. Go to System Settings > Add-ons
  3. Toggle the slider next to the "Unknown Sources" option
  4. Accept the warning message that pops up

Adding Repository and Installing Video Add-on(currently Neptune Rising)

  1. Click on the gear icon to enter System Menu
  2. Go to File Manager > Add Source
  3. In the top box tpye in the following URL exactly as it appears: http://blamo.teverz.com/
  4. Name the new repository "Blamo"
  5. Click "OK"
  6. Go back to the Kodi Main Menu
  7. Select Add-ons
  8. Click the open box icon at the top of the menu bar
  9. Choose "Install from Zip File"
  10. Select "Blamo" and install repository file
  11. Go back to the Add-ons menu and click on the open box again
  12. Choose "Install from repository"
  13. Select Blamo, should be at the top of the list
  14. Click "Video add-ons" near the bottom, select Neptune Rising and install
  15. Done
