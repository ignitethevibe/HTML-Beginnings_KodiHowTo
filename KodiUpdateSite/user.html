Kodi User Walkthrough

How to Access and Use Kodi on your Amazon Firestick

  • If the Kodi app is not on the Home Screen then from the top scroll all the way right to Settings
  • From settings scroll right until you reach Applications(click)
  • In Applications scroll down to Manage Applications(click)
  • Scroll down and select Kodi, then select Launch
  • You've now opened Kodi
  • In Kodi scroll down to the Add-Ons tab
  • Then select desired Video Add-On
  • Within [video add-on] you are given options towards how you would like to search for media
  • For Movies select Movies, for TV Shows select TV Shows
  • Once you've found what you want to watch, select it and it will begin to load streams
  • Once Loaded it will present a list of available streams, better streams are towards the top
  • *If you select a stream and get a message about some service just click cancel and it will continue down the list until it finds a playable stream
  • __
  • At the bottom there is always a search option
  • - I recommend downloading the FireTV app on your phone to make it easier by giving you a keyboard to type

Problems/Issues Fix

Is it Freezing or Not Loading?

Go to the FireStick Settings (2 Options)

Option 1(Kodi App Restart)

  • Go to Applications
  • Select Manage Applications
  • Scroll down and select Kodi
  • Select Force Stop
  • Then Select Launch App

Option 2(FireStick Restart)

  • In Settings go to Device
  • Scroll down and select Restart
